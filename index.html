Signal-strength independent filters (SSI or sometimes, SS2) are not one-wide-tilable, and each filter has to be tiled with compatible filters. This (incomplete, but useful) reference shows compatible designs by selecting (clicking on) one of the designs, invalid designs are ghosted. Multiple chests indicate that there are multiple potential positions for the output chest, only use one.

If there is some mistake, or, you would like to add something, please message me on Discord: dzreams#5634. Data was originally collated by KikuGie#7003. This google sheet shows the same data but in matrix format.
